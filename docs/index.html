Documentación de la API

Esta es la documentación de la API.

Introducción

Bienvenido a la documentación de la API. Aquí encontrarás toda la información necesaria para interactuar con nuestros servicios.

Autenticación

Para acceder a la API, necesitas autenticarte utilizando un token de acceso. Asegúrate de incluir el token en el encabezado de tus solicitudes.

Endpoints

A continuación se detallan los endpoints disponibles:

  • /api/v1/usuarios - Obtiene una lista de usuarios.
  • /api/v1/usuarios/{id} - Obtiene los detalles de un usuario específico.
  • /api/v1/productos - Obtiene una lista de productos.
  • /api/v1/productos/{id} - Obtiene los detalles de un producto específico.
  • /api/v1/pedidos - Crea un nuevo pedido.
  • /api/v1/pedidos/{id} - Obtiene los detalles de un pedido específico.
